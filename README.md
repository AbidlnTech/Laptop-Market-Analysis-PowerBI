# Laptop-Market-Analysis-PowerBI
Power BI dashboard project analyzing laptop market trends using data cleaning, visualization, KPI cards, slicers, and brand/core-wise insights.

# Laptop Market Analysis Dashboard

## Overview
This project analyzes laptop market trends using Power BI.

## Tools Used
- Power BI
- Excel
- DAX

## Workflow
1. Collected raw laptop dataset
2. Cleaned and filtered the data
3. Created interactive dashboard in Power BI
4. Generated insights using visualizations

## Features
- KPI Cards
- Brand Analysis
- Core-wise Analysis
- Price Comparisons
- Interactive Filters

## Project Structure
- Raw_Data → Original dataset
- Cleaned_Data → Processed dataset
- PowerBI → Dashboard file
- Screenshots → Dashboard images
